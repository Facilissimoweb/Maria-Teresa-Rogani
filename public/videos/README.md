# Cartella Video per la sezione Hero

Carica qui il tuo file video per lo sfondo della sezione Hero.

## Istruzioni:
1. Rinomina il tuo file video in `hero-bg.mp4`.
2. Salvalo in questa cartella (`/public/videos/hero-bg.mp4`).
3. Il sito lo rileverà automaticamente e lo riprodurrà in loop come sfondo della sezione Hero, integrandosi elegantemente con le particelle arancioni in primo piano!

## Formato Consigliato:
* Formato: **MP4** (`.mp4`)
* Risoluzione: **1080p** (1920x1080) o inferiore (consigliato per tempi di caricamento rapidi)
* Durata: Un loop breve e fluido di **10-30 secondi**
* Dimensione: Inferiore a **10MB** per non rallentare l'accesso iniziale al sito
